# Neovim Configuration
## Requirements

In addition to `kortla-dev/config.nvim`, this configuration requires the following external components:

- **Neovim**
- **tree-sitter-cli**: can be installed with cargo or npm 
- **dotnet**
- **dotnet sdk**
- **Python**
- **Node.js**
- **Tex Live**
- **latexmk**
- **mupdf**
